# Zexo Studio (Check out our other services too!: zexo.pro)

lightweight local LLM client. fast, clean, not bloated.

runs models locally. no dependencies. thats it.

## why

- lm studio is heavy
- ollama is just a terminal
- zexo studio is the middle ground
